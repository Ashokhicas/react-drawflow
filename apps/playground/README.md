# React Drawflow Playground

Interactive demo application for React Drawflow library.

## Development

```bash
# Start development server
pnpm dev

# Build for production
pnpm build
```

The playground app demonstrates all features of React Drawflow including:

- Node creation and management
- Connection system
- Multi-select
- Undo/Redo
- Module switching
- Theme toggling
- Orientation switching
- Export/Import
- And more...

Visit `http://localhost:3000` to see the playground in action.
